# Security Utilities Toolkit

A collection of Python scripts for **threat intelligence**, **malware analysis**, and **security research**.  
These tools help automate common tasks such as URL/hash lookups using VirusTotal, basic PE file inspection, and other defensive utilities.

This repository is intended for **educational, defensive security, and research purposes only.**  
No malware samples are included.
